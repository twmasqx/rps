# Nexus Vision

**أداة إدارة شبكة احترافية** — لشبكاتك المملوكة. واجهة بتصميم Cybersecurity تعرض الأجهزة بالعلامة التجارية والموديل.

---

## مميزات التطبيق

### 🔍 فحص الشبكة
- **مسح ARP تلقائي** — كشف الأجهزة المتصلة على الشبكة المحلية
- **عرض IP و MAC** — معرفة عنوان كل جهاز
- **التعرّف على الشركة** — تحديد المصنع (Apple, Samsung, Huawei, Xiaomi...) عبر قاعدة OUI

### 📡 رادار مرئي
- **عرض تفاعلي** — تمثيل الأجهزة على دائرة رادار مع شعاع مسح متحرك
- **توهج عند المرور** — تأثير بصري عند مرور الشعاع بالقرب من الجهاز
- **أثر الحركة (Motion Trail)** — خط يتبع حركة كل جهاز
- **أيقونات حسب النوع** — تمييز أجهزة Apple عن Android

### 📋 قائمة الأجهزة
- **MDList احترافية** — قائمة أنيقة تعرض كل جهاز مع IP و MAC والنوع
- **النقر للتفاصيل** — عرض النطاقات التي زارها الجهاز
- **تحديث فوري** — تحديث تلقائي عند اكتشاف أجهزة جديدة

### 📊 مراقبة النطاقات
- **Monitor** — شاشة لرصد النطاقات (DNS/SNI/Host) التي زارها كل جهاز
- **تصدير CSV** — حفظ البيانات لتحليلها لاحقاً
- **سجلات JSON** — حفظ سجل المسح والطلبات

### 🛡️ أزرار الإجراءات
| الزر | الوظيفة |
|------|---------|
| **Scan** | مسح فوري مع شريط تقدم |
| **Intercept** | فتح Monitor للروابط |
| **Kick** | طرد فعلي (ARP) عند Root، أو محاكاة |
| **Monitor** | رصد DNS/SNI/HTTP - سجل حي يتحدث كل 2 ثانية |
| **Kill Switch** | إيقاف طوارئ لكل العمليات |

### 📤 إرسال عن بُعد (اختياري)
- انسخ `nexus_config.json.example` إلى `nexus_config.json`
- أضف `api_url` أو `telegram_bot_token` + `telegram_chat_id` لإرسال الروابط المكتشفة

### 🎨 التصميم
- **ثيم داكن Cybersecurity** — ألوان سيان/أخضر تقنية
- **واجهة KivyMD** — مكونات Material Design احترافية
- **متجاوب مع الجوال** — يتكيف مع الشاشات الطولية
- **شريط تقدم** — يظهر حالة الفحص

## التشغيل

### على سطح المكتب
```bash
pip install -r requirements.txt
python main.py
```

### بناء APK لأندرويد
```bash
buildozer -v android debug
```

### عبر GitHub Actions
- يتم البناء تلقائياً عند كل `push`
- تحميل APK من تبويب **Actions** → **Artifacts** → `nexusvision-apk`
